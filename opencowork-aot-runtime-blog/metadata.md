## Short Summary
OpenCoWork 不是另一个只能对话的 AI 窗口，而是一个真正能进入工作现场的开源桌面多智能体协作平台：它能读写文件、执行命令、连接 SSH、挂 MCP、接入办公消息系统，还在迁移到 .NET AOT Runtime 后，把性能体验拉到了一个更轻、更稳的新水平。

## Long Intro Blurb
很多 AI 产品的问题，不是模型不够强，而是离真实工作太远：它看不到你的项目、碰不到你的终端、接不上你的远程环境，也很难在多任务和高强度使用时保持足够轻盈。OpenCoWork 想解决的，正是这件事。它把多智能体协作、本地工具、SSH、MCP、定时任务、消息系统接入和多 provider 统一配置放进一个桌面工作台里；更重要的是，在 .NET AOT Runtime 方向上，它还把 CPU 与内存占用压得更克制，让“Agent 真能天天开着用”这件事，开始变得现实。
